# Attor Public Assets

Public assets for the [Attor](https://attor.ai) platform.

## Contents

- `assets/icons/` - Application icons for Authentik
- `assets/branding/` - Logo, favicon, backgrounds

## Usage

These assets are served via GitHub's raw content CDN:

```
https://raw.githubusercontent.com/flengure/attor-public/main/assets/icons/<filename>
https://raw.githubusercontent.com/flengure/attor-public/main/assets/branding/<filename>
```

## Sync

This repo is automatically synced from the private Attor repository via GitHub Actions.
